# Meccha Box ESP 更新紀錄

日期：2026-06-17

## 目前狀態

專案已重構為外部唯讀 Box ESP，不再繪製玩家骨架。

目前繪製基準改為 `actor + 0x418` 的 body mesh，讀取 `CachedComponentSpaceTransforms + 0x928` 後，使用 transform array 的第 0 筆作為人物腳底中心 anchor。這個 anchor 對應畫面中最穩定、位於人物正中間腳底的點。

方框計算方式：

1. 讀取 body mesh component。
2. 讀取 transform array 第 0 筆作為腳底 anchor。
3. 將腳底 world position 投影到螢幕。
4. 以 `footWorld.Z + Box world height` 估算頭頂位置並投影。
5. 用腳底與頭頂投影點組成 2D 方框。
6. 可選擇繪製腳底 anchor dot。

## 非唯讀功能方向

目前主線仍維持外部唯讀。可安全擴充的功能方向：

- 距離顯示。
- 方框顏色分類。
- 腳底 anchor 顏色與大小調整。
- 畫面邊緣 off-screen indicator。
- 2D radar。
- 目標列表。
- 設定檔保存與載入。
- hotkey 控制 UI / ESP / dot / box。

不建議直接加入 live game 寫記憶體、注入、hook、移動控制或狀態修改功能；若需要非唯讀互動，應限定在自有測試 build、授權 mod/debug 環境或官方提供的插件入口。
